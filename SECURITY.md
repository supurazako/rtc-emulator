# Security Policy

## Supported Versions

This project is currently in early development.
Security fixes are provided for the latest development code on `main`.

| Version | Supported |
| ------- | --------- |
| main | ✅ |
| older commits / forks | ❌ |

## Reporting a Vulnerability

Please do **not** open public GitHub issues for security vulnerabilities.

Report vulnerabilities by email to:
`rtc.emulator.dev@gmail.com`

Please include:

- A clear description of the issue
- Steps to reproduce (if available)
- Potential impact
- Any suggested fix or mitigation

We will acknowledge receipt as soon as possible and investigate the report.
